## 一句话

搜 GitHub 20k+ stars 的 agent 技能项目 → 你选 → 确认位置 → 自动安装到 Hermes/Cursor/Claude Code/Codex。

## 使用

```
"skill-scout" 或 "找高星 skill"
→ 搜索 → 列表 → 你选项目 → 你选位置 → 安装
```

支持：Hermes / Cursor / Claude Code / Codex
